## üìå Project Overview
**Vehicle Configurator** is a **B2B web-based portal** designed for vehicle leasing operations. The project is sponsored by a **leasing company**, while the end users are **rental car companies** that require a fleet of vehicles on lease.  

The system enables users to **select, configure, and price vehicles dynamically** based on predefined rules for each model. The total cost changes in real time according to selected configurations. Upon successful completion of a transaction, a **detailed invoice is generated in PDF format and sent via email**.
